# 德州扑克AI训练软件 - 完整开发计划

## 项目概述
AI原生德州扑克水平训练软件，核心壁垒：个性化AI教练+实时弱点诊断闭环

## 技术栈
- 前端：React 18 + TypeScript + Tailwind CSS + ShadCN UI
- 后端：Node.js + Express
- 数据库：Supabase PostgreSQL
- 认证：Supabase Auth
- 部署：Vercel (前端) + AWS Lambda (AI计算)
- AI层：OpenAI GPT-4o + RAG知识库 + 轻量级GTO Solver

## 第一阶段：MVP（第1-4周）

### 核心功能1：手牌历史导入与AI分析
- 支持平台：PokerStars、GG Poker、888、PartyPoker、Winamax
- 手牌格式解析器（各平台格式转换）
- AI自动标注：
  - EV损失计算
  - 位置错误识别
  - 范围错误识别
- 自然语言解释（基于权威教材）
- 个人错误统计报告

### 核心功能2：智能错题训练
- 根据错误报告自动生成同类型题目
- 每题即时详细解释
- 无限量题目生成
- 正确率追踪

### 核心功能3：翻前范围训练
- 覆盖所有位置（UTG/MP/CO/BTN/SB/BB）
- 不同筹码深度（20BB/50BB/100BB/200BB）
- 开池/3bet/4bet/跟注范围
- 自动记录正确率
- 弱项强化训练

## 第二阶段：v1.0（第5-8周）
